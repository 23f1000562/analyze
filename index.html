Welcome to Our Data App

Explore data processing with Python and Tailwind CSS.

Python Script (`execute.py`)

Our `execute.py` script efficiently processes structured data. It's built with robustness in mind, handling potential data inconsistencies to ensure reliable output.

It reads `data.csv`, performs aggregations, and outputs the result in JSON format.

Data Source (`data.csv`)

The core data for this application is provided in `data.csv`, converted from `data.xlsx`. This standardized format ensures compatibility and ease of processing.

It serves as the input for our Python processing script.

Continuous Integration & Deployment

This project features a robust CI/CD pipeline using GitHub Actions. It ensures code quality with Ruff and automates the generation and publication of `result.json` to GitHub Pages with every push to the main branch.
